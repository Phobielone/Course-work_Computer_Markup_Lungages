Russian modder added a message system to Skyrim and Fallout 4 - like in Dark Souls and Elden Ring

28.04.2022 | 16:40

The Building Bridges addons for The Elder Scrolls V: Skyrim and Fallout 4 were developed by a well-known Russian modder named TommInfinite. Expansions add message mechanics to RPGs, like in the Dark Souls and Elden Ring series. It allows players to leave messages anywhere in the game world, such as warning of dangers.

The modder came up with a rating system for messages. At first, messages are limited to 115 characters. However, if the player gets enough positive ratings, they will be able to write more detailed messages.

In order to leave a message, you need to press the key with the English "B" and enter the text. In the game world, the messages look like notes, the appearance of which in Skyrim, if desired, can be customized through the configuration file. In addition, the author of the mod provided the ability to quickly disable their display without having to remove the addon.

Extensions support most other mods for games, including those related to text input. The author of the addon also provided a system of complaints, which will allow you to delete messages containing insults, racist or political statements.

The Building Bridges mod for Skyrim and Fallout 4 is available for download at Nexus Mods. To run the extension, you must first download the plugins SUP SKSE for Skyrim and SUP F4SE for Fallout 4. After that, you need to request an API key for the mod in your Nexus Mods account settings. Building Bridges installation instructions are available on each extension's page.
