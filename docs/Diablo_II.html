Diablo II: Resurrected launches its first ranked season

22.04.2022 | 22:38

As previously promised, Blizzard has launched the first ranked season of Diablo II: Resurrected, where players will be able to test their skills in special modes and compete for the title of the best. The experience earned by gamers will be displayed in separate tables, and the data will be reset after the end of the season.

Note that earlier the game has already received a large-scale patch 2.4, which introduced a number of changes to the balance of classes, added new graphics modes and traditionally contains a number of bug fixes.

We also remind Blizzard fans about the upcoming broadcast, which will take place on Tuesday and will be dedicated to the new mobile game in the Warcraft universe.
